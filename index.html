How do I get a token?
  1. If you own the shared repo: open Fine-grained tokens → Generate new token → Repository access → Only select repositories → pick the shared repo → Permissions → Repository → Contents: Read and write.
  2. If you were added as a collaborator: a fine-grained token can't reach a repo owned by someone else, so use a classic token → tick the repo scope → Generate token.
  3. Copy the token string and paste it above. It's saved only in this browser.
